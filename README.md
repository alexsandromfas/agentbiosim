# smart_bacteria
Simulação de bactérias autônomas com CUDA e SFML, onde cada agente é controlado por uma pequena rede neural que decide sua velocidade e direção a partir de estímulos do ambiente. As bactérias competem por comida, se reproduzem quando acumulam energia e morrem quando falham, permitindo observar um processo de seleção natural emergente em tempo real.
